# 运行指南

- 依赖安装
  //如果没有安装过 pnpm
  `npm i -g pnpm`
  `pnpm i `
- 运行打包微信开发者工具命令
  `pnpm run dev:mp-weixin `
- 在微信开发者工具中打开 dist 项目下的 mp-weixin 文件夹，点击运行即可
